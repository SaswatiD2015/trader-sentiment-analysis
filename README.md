# Trader Behavior Analysis Using Fear & Greed Index

This project analyzes how Bitcoin market sentiment affects trader
behavior and profitability using historical trading data.

## Dataset
1. Fear and Greed Index
2. Hyperliquid Trader Historical Data

## Analysis Performed
- Trader profitability vs sentiment
- Trade size analysis
- Trade activity patterns
- Profit/Loss distribution

## Tools Used
Python
Pandas
Seaborn
Matplotlib
Jupyter Notebook
